## 🇧🇷 Versão em Português

Oi! Esse é o repositório do meu primeiro projeto/game indie, **Contos em Branco**. É uma visual novel de mistério, sobrevivência com uma pitada de romance focada em múltiplas rotas e escolhas que realmente importam.

Mais do que um jogo, esse projeto é o meu "laboratório" pra colocar em prática o que eu estudo em Engenharia de Software. Em vez de só fazer código pra passar nas matérias da faculdade, eu resolvi juntar tudo aqui.

### 🛠️ Stack
* **O Jogo (Unity + C#):** Motor principal. Cuida da interface, dos diálogos e da árvore de decisões das rotas.
* **O Site (React + Next.js + TypeScript):** Dashboard web. Onde o jogador faz login e acessa o inventário fora do jogo.
* **O Cofre (Azure PlayFab):** Backend e banco de dados. Guarda os saves na nuvem e gerencia a economia com segurança.

## 🇺🇸 English Version

Hi! This is the repository for my first indie game/project, **Contos em Branco** (*Blank Tales*). It's a mystery and survival visual novel with a dash of romance, focused on multiple routes and choices that actually matter.

More than just a game, this project is my "laboratory" to put into practice what I study in Software Engineering. Instead of just writing code to pass my college classes, I decided to bring it all together here.

### 🛠️ Tech Stack
* **The Game (Unity + C#):** Main engine. Handles the UI, dialogues, and the branching narrative.
* **The Site (React + Next.js + TypeScript):** Web dashboard. Where the player logs in and accesses their inventory outside the game.
* **The Vault (Azure PlayFab):** Backend and database. Secures cloud saves and safely manages the economy.
